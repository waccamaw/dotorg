Layout

Max Width

Content containers use a maximum width of --max-width: 900px for optimal readability.

Spacing

The design uses consistent spacing:

  • Small gaps: 0.5rem - 1rem
  • Medium gaps: 1.5rem - 2rem
  • Large gaps: 3rem - 4rem

Border Radius

Rounded corners for visual softness:

  • Small elements (buttons, tags): 4px
  • Cards and containers: 8px
  • Search input: 20px
  • Circular elements: 50%

Usage Guidelines

Accessibility

  • All colors meet WCAG AA contrast standards for text
  • Interactive elements have clear focus states
  • Images include alt text
  • Semantic HTML structure throughout

Responsive Design

  • Mobile-first approach with breakpoint at 768px
  • Flexible layouts using CSS Grid and Flexbox
  • Logo and navigation adapt for smaller screens

CSS Variables

All colors and key values are defined as CSS custom properties in :root for easy theme customization and maintenance.
